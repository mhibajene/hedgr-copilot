# Frontend App

This is the main frontend application for the Hedgr Copilot project.

## Development

Run the development server:

```bash
pnpm dev
```

## Testing

Run the test suite:

```bash
pnpm test
```

## E2E Testing

Run end-to-end tests:

```bash
pnpm e2e:ci
```
