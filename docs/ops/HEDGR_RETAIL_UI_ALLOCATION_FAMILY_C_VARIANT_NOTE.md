## 4. Why a variant note is needed

Family C has already been selected as the **active refinement family**, but its internal cue range still needs bounded comparison.

This note is needed to:

- keep the next comparison **small** and same-family
- avoid uncontrolled internal Family C drift
- define which internal variants are worth testing next
- preserve **row-first clarity** and **calm subordination**

This remains governance-safe and non-implementing: the output is critique discipline, not product instruction.

---

## 5. Variant framing principle

**All Family C variants should preserve the same underlying pattern family while varying only the strength and role of the secondary structural cue.**

Across all variants:

- row clarity remains **primary**
- cue remains **secondary**
- the comparison tests **internal cue range**, not new family invention

## 6. Variant set

### 6.1 Variant C1 — Minimal hybrid

- Lightest cue expression inside Family C.
- Closest to Family A calmness while still retaining a bounded structural cue.
- Strongest anti-busyness posture.
- **Risk:** may collapse back toward A-like passivity if the cue becomes too recessive.

### 6.2 Variant C2 — Balanced hybrid

- Likely current centerline variant inside Family C.
- Aims to balance row clarity with clear but modest structural reinforcement.
- Strongest candidate for next comparison pass.
- **Risk:** may quietly become the assumed winner if not kept critique-only.

### 6.3 Variant C3 — Stronger hybrid stress test

- Upper-bound Family C stress test.
- Useful for learning when cue strength starts to feel busy, precise, or hierarchy-creeping.
- Not intended as a default direction.
- **Risk:** drifts toward B-like precision implication or mid-layer salience creep.

---

## 7. What changes across variants

Allowed to vary across C1 / C2 / C3:

- cue visibility and cue strength
- cue-to-row balance
- degree of structural emphasis

Does **not** vary:

- hierarchy placement
- labels
- broader screen anatomy
- allocation informational posture
- any engine or protocol meaning

This distinction is strict: only bounded internal cue-range pressure is tested.
