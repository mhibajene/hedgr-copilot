## 4. Why a same-family micro-test is needed

Family C has already been selected as the active family, so the question is no longer which family wins.

The purpose of this micro-test is to isolate internal Family C behavior on:

- cue strength
- row/cue balance
- calmness retention
- anti-busyness behavior
- anti-precision behavior

This remains governance-safe and non-implementing: it is critique discipline for bounded refinement only.
