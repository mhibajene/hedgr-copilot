## 5. Current role of C1 / C2 / C3

### 5.1 C1 — Minimal hybrid

C1 remains the calm lower bound and anti-busyness control inside Family C. It is a useful honesty and restraint reference.

Its limit is structural softness: if cueing falls too far, C1 stops carrying enough structural gain to hold Family C’s purpose.

### 5.2 C2 — Balanced hybrid

C2 remains the strongest current internal centerline candidate. It is the best reference point for acceptable cue range and for balancing row clarity with structural grasp.

C2 still requires governance discipline because it can quietly be mistaken for default truth rather than a bounded interpretive treatment.

### 5.3 C3 — Stronger hybrid stress test

C3 remains an upper-bound failure detector. It is useful because it exposes where Family C becomes too assertive.

It is not a default direction. It carries the highest risk of precision implication, busyness, and hierarchy creep.

## 6. Acceptable Family C cue range

The likely acceptable Family C cue envelope sits closer to C2 than C3, while still retaining more structure than C1.

Inside this envelope, row-first clarity remains primary and cueing remains clearly interpretive rather than exact. The boundary is set by restraint, not by expressiveness.
