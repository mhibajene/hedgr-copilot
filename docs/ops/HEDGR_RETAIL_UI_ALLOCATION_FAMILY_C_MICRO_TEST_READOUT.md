## 4. What the micro-test confirmed

- Family C can be refined internally without reopening family selection.
- Cue-range effect became easier to judge because family and hierarchy stayed fixed.
- Stronger cueing can improve structural grasp, but quickly raises precision and salience risk.
- Weaker cueing preserves calmness, but can underdeliver structural gain.
- The key question is where Family C stops being restrained and starts becoming busy or too exact.
